Add Money From Bank Account

Account Holder's Name should contain only alphabets and can have spaces.
Mobile Number should be exactly 10 digits.
Account Number should be exactly 12 digits.
IFSC Code should be in the format Ex:ABC1234567.
ATM Card Number should be exactly 16 digits.
ATM PIN must be exactly 4 digits.
Receiver ID must be positive.
Amount must be greater than or equal to 100.
Amount must be less than or equal to 100000.
